How the Web Works
The web consists of people and computers communicating with each other via 3 main components:
* browser - computer/human interface for the web; i.e., Mozilla, Internet Explorer, Safari, Chrome
* internet - the world's largest computer network
* servers - host files with content including text, audio, images and video
HTML
HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". For more information about current HTML standards check out the w3.org page.
Tags and Elements
HTML elements consist of HTML tags + content. The tags contain formatting information telling the browser how the associated content should appear and function; i.e., bold, italicized, simple content or a link to another webpage. Most HTML tags have the format <opening tag> content </closing tag> although some, like the image source tag, are designed without a closing tag.
Why Computers are Stupid
Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause unintended behavior in a program.
Inline vs Block Elements
HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them, imparting additional formating attributes.
Box sizing
Boxes have 4 characteristics. Working from the inside out they are content, padding, border and margin. It is best to set sizes as a percentage rather than in pixels, although a maximum size may be specfied in pixels.
